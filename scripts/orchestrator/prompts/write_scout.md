# 砚·选股遴选报告 撰写指南

你正在撰写一份 A 股选股遴选报告。这不是收盘复盘，是专门的**选股交付物**。

## 报告结构（严格按此顺序）

### 1. 数据覆盖说明
列出本次选股流程中每个数据模块的可用状态（✅/❌/⚠️）。
对缺失数据明确标注"因缺失该数据，以下结论置信度下调"。

### 2. 市场环境判断
- 涨跌广度（涨/跌/平/涨跌比）
- 情绪温度（极热/偏热/中性/偏冷/极冷）
- 当前适合哪种操作（短线/中线/长线/观望）
- 大盘位置（上证/深成指/创业板指涨跌幅）

### 3. 短线候选（1-5 天）
每只必须包含：
- 代码+名称+所属行业
- 入选理由（从五维评分中取最强的 2-3 个维度）
- 入场价区间 / 止损位 / 目标位
- 仓位建议（占总组合比例）
- 主要风险
- 置信度（高/中/低）

### 4. 中线候选（1-4 周）
格式同上，但重点说明：
- 产业逻辑 / 基本面支撑
- 催化剂时间窗口

### 5. 长线候选（1-12 个月）
格式同上，但重点说明：
- 估值位置 / 安全边际
- 成长阶段 / FFScore

### 6. 板块轮动候选
从 stage2b 结果中提取：
- 哪些景气行业有尚未被发现的机会
- 为什么这些行业是当前主线

### 7. 资金流传导分析
从 stage2c/2d 结果中提取：
- 检测到的传导信号（政策/龙虎榜集中/上下游）
- 传导候选的因果链解释
- Claude 推理的置信度排序（如有）

### 8. 被拒绝的候选
列出评分较高但最终没入选的股票，说明拒绝原因。

### 9. 如果我错了
- 最可能错在哪里
- 什么信号会推翻当前判断

### 10. 下次复核
- 建议下次选股时间
- 需要重点跟踪的变量

## 硬规则
- 每只入选候选至少 3 个维度评分 ≥ 60
- 大盘广度 < 40% 时短线候选不超过 2 只
- 必须有止损位
- 不要用"综合来看""总体而言"等空话
- 所有数字引用必须标注来源
- 置信度 = 低的不入选最终名单
