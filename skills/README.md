# Skills

本目录存放技能的符号链接，指向 OpenClaw 的 `skill-library/`。

## 当前启用的技能

- agent-browser — 浏览器自动化
- akshare-stock — AkShare A股数据
- exploratory-data-analysis — 探索性数据分析
- hedgefundmonitor — 对冲基金持仓监控
- market-research-reports — 券商研报采集
- markitdown — 文件转 Markdown
- plotly — Plotly 可视化
- polars — Polars 数据处理
- statistical-analysis — 统计分析
- summarizer — 文本摘要
- usfiscaldata — 美国财政数据
- xiucheng-self-improving-agent — 自改进 Agent

## 配置方式

```bash
# 将 skill-library 中的技能链接到本目录
ln -s ../../skill-library/<skill-name> skills/<skill-name>
```
